# Website Portofolio Pribadi

Website portofolio pribadi yang dibuat menggunakan React dan Tailwind CSS untuk menampilkan profil, kemampuan, proyek, dan informasi kontak dalam satu halaman yang modern dan responsif.

---

## Fitur

* Tampilan responsif untuk semua ukuran layar
* Loading screen interaktif
* Animasi scroll
* Navigasi single page menggunakan React Router
* Halaman project
* Form contact dengan EmailJS
* Animasi modern menggunakan GSAP dan AOS
* Elemen 3D interaktif

---

## Bagian Website

### Home

Halaman utama yang berisi perkenalan singkat dan gambaran umum tentang diri saya.

### About Me

Berisi informasi tentang latar belakang sebagai mahasiswa Informatika, minat di bidang web development, serta proses belajar teknologi.

### Skills

Menampilkan kemampuan teknis seperti bahasa pemrograman, framework, dan tools yang digunakan dalam pengembangan web.

### Projects

Berisi kumpulan proyek yang pernah dibuat, baik proyek pribadi maupun tugas kuliah.

### Contact

Berisi informasi kontak untuk menghubungi saya melalui email atau media sosial.

## Cara Menjalankan Project

Clone repository:

```bash
git clone https://github.com/Lodi7/MyPortofolio
```

Masuk ke folder project:

```bash
cd MyPortofolio
```

Install dependency:

```bash
npm install
```

Jalankan project:

```bash
npm run dev
```
